# python-auto-capture

오전 9시~오후 6시, 매시 10분마다 자동으로 스크린샷을 캡처하는 macOS용 프로그램

## 사용법

```bash
python3 capture.py
```

종료: `Ctrl + C`

## 저장 위치

```
Screenshots/
├── 2025-09-17/
│   ├── Screenshot 2025-09-17 at 09.10.00 AM.png
│   └── Screenshot 2025-09-17 at 10.10.00 AM.png
└── 2025-09-18/
    └── ...
```

## 요구사항

- macOS
- Python 3.x
- 스크린 기록 권한 필요
